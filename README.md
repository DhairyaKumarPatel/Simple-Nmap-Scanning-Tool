## Run Locally

Clone the project

```bash
  git clone https://github.com/DhairyaKumarPatel/Simple-Nmap-Scanning-Tool
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  apt install python3-nmap
```

Run the script

```bash
  python3 nmap_scanner.py
```
